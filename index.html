The Optimization Zoo

A guide to the diverse world of optimization problems

Welcome to the Optimization Zoo! This resource aims to help developers and enthusiasts understand and categorize the vast landscape of optimization problems. Whether you're dealing with resource allocation, scheduling, design, or data analysis, you'll find a problem type that resonates with your challenge.

I. Fundamental Optimization Paradigms

These categories represent core mathematical frameworks for defining and solving optimization problems based on the nature of their objective functions, constraints, and variables.

A. Continuous Linear & Non-Linear Optimization

Problems where all decision variables are continuous, ranging from purely linear to complex non-linear forms.

Problem Name Description Primary Paradigm Related / Inherits From
Linear Programming (LP) Optimizing a linear objective function ($75x_1 + 50x_2 + 35x_3$) subject to linear equality and inequality constraints. All variables are continuous. LPs are inherently convex, making them intrinsically easier to solve (e.g., via Simplex or Interior Point methods), with solutions always at 'corner points'. Continuous Linear (Parent) N/A
Quadratic Programming (QP) Optimizing a quadratic objective function (e.g., $2x_1^2 + 3x_2^2 + 4x_1x_2$) subject to linear constraints. Variables are continuous. QPs can be convex (single minimum, like portfolio optimization) or non-convex ("saddle" shape, multiple local optima), significantly impacting solvability. Solved by specialized Simplex/Interior Point extensions or general NLP methods. Continuous Quadratic LP (linear constraints, quadratic objective), NLP (special case)
Non-Linear Programming (NLP) Optimizing an objective function and/or constraints that are non-linear. Generally more complex to solve than linear or quadratic problems, as they can be non-convex and involve multiple feasible regions and local optima. Continuous Non-Linear (Parent) N/A
Quadratically Constrained Quadratic Programming (QCQP) Optimizing a quadratic objective function subject to quadratic constraints. A generalization of QP and a specific type of NLP, found in signal processing and engineering design. Continuous Non-Linear (Quadratic Form) NLP (quadratic objective & constraints), QP (more generalized)

B. Discrete & Mixed-Integer Optimization

Problems where some or all decision variables must take integer or binary values, often used for modeling discrete choices and resource allocation.

Problem Name Description Primary Paradigm Related / Inherits From
Integer Programming (IP) An extension of LP where some or all of the variables are restricted to be integers. Essential for modeling indivisible items or yes/no decisions. Discrete Linear (Parent) LP (adds integer restriction)
Binary Integer Programming (BIP) A special case of IP where variables are restricted to be either 0 or 1, representing "yes/no" or "on/off" decisions. Crucial for combinatorial problems. Discrete Linear (Binary) IP (special case of integer variables)
Mixed-Integer Linear Programming (MILP) Combines LP and IP, allowing for both continuous and integer variables within a linear framework. Widely used in planning and scheduling. Mixed-Integer Linear LP, IP (combines continuous and integer variables)
Mixed-Integer Non-Linear Programming (MINLP) The most general type, including both continuous and integer variables, with non-linear objectives and/or constraints. Very challenging to solve optimally due to the combination of non-convexity and discrete variables. Mixed-Integer Non-Linear NLP, MILP (most general form)

C. Constraint-Based Paradigms

Problems focused on finding any valid assignment of values to variables that satisfies a given set of conditions, rather than optimizing a specific objective function.

Problem Name Description Primary Paradigm Related / Inherits From
Constraint Satisfaction Problem (CSP) Finding a state or a set of values for variables that satisfies a given set of constraints, without necessarily optimizing an explicit objective function. Common in AI for puzzles and logical problems. Constraint Programming (Parent) N/A

II. Canonical Problem Formulations & Models

These problems represent particular mathematical structures or models often used across different optimization domains, sometimes serving as building blocks or theoretical equivalents for other problems.

Problem Name Description Primary Paradigm Related / Inherits From
Second-Order Cone Programming (SOCP) Optimizing a linear objective function over the intersection of affine subspaces and second-order cones. Both Linear Programming (LP) and convex Quadratic Programming (QP) are special cases of SOCP, making it a powerful general convex optimization framework. Solved efficiently by interior point methods. Conic Optimization LP, Convex QP (generalized form)
Quadratic Unconstrained Binary Optimization (QUBO) Optimizing a quadratic objective function where variables are binary (0 or 1) and there are no explicit constraints. Many combinatorial problems can be mapped to QUBO, making it suitable for quantum annealing. Canonical Binary Ising Model, BIP
Ising Model A mathematical model representing a lattice of discrete variables (spins) in one of two states ($+1$ or $-1$). Closely related to QUBO and used in statistical mechanics and combinatorial optimization, particularly with quantum computing. Canonical Binary QUBO (equivalent formulation)
Boolean Satisfiability (SAT) Determining if there exists an assignment of truth values to variables in a Boolean formula such that the formula evaluates to true. A foundational problem in computational complexity and logic. Canonical Logic CSP (specific type of CSP)
Set Cover Problem Given a set of elements and a collection of sets whose union is the universal set, find the minimum number of sets whose union is still the universal set. Classic problem in resource allocation. Canonical Combinatorial BIP (can be formulated as such)
Dominating Set Problem Finding the smallest subset of vertices in a graph such that every other vertex not in the subset is adjacent to at least one vertex in the subset. Relevant in network design. Canonical Graph Vertex Cover, Set Cover
Maximum Independent Set Problem Finding the largest set of vertices in a graph such that no two vertices in the set are adjacent. Has applications in scheduling and conflict resolution. Canonical Graph Graph Theory, Vertex Cover (complement)

III. Combinatorial Optimization Problems (Graph-based & Discrete)

These involve finding an optimal object from a finite set of objects, often permutations, combinations, or graphs, typically characterized by discrete choices.

A. General Combinatorial Problems

Problem Name Description Primary Paradigm Related / Inherits From
Knapsack Problem Given a set of items, each with a weight and a value, determine the subset of items to include in a collection so that the total weight is within a given limit and the total value is maximized. A classic resource allocation problem. Combinatorial IP, Dynamic Programming
Set Packing Problem Given a collection of finite sets, find the largest sub-collection of sets such that any two sets in the sub-collection are disjoint. Applications in scheduling and resource allocation. Combinatorial IP, Set Cover
Assignment Problem Assigning agents to tasks (or resources to jobs) such that each agent is assigned to exactly one task and each task is assigned to exactly one agent, optimizing a total cost or benefit. Common in operations research. Combinatorial LP, Network Flow

B. Graph-Based Combinatorial Problems

Problem Name Description Primary Paradigm Related / Inherits From
Max Cut Problem Partitioning the vertices of a graph into two sets such that the sum of weights of edges connecting vertices in different sets is maximized. Fundamental in clustering and network analysis. Graph Optimization Graph Theory, QUBO
Min Cut (Max Flow) Problem Finding a set of edges whose removal separates a source vertex from a target vertex in a flow network, such that the total capacity of these edges is minimized. This is dual to the Max Flow problem, critical in network reliability and logistics. Graph Optimization Graph Theory, Network Flow
Graph Coloring Problem Assigning colors to vertices of a graph such that no two adjacent vertices in a graph share the same color, using the minimum possible number of colors. Applications include scheduling and frequency assignment. Graph Optimization Graph Theory, CSP
Traveling Salesman Problem (TSP) Given a list of cities and the distances between each pair of cities, find the shortest possible route that visits each city exactly once and returns to the origin city. A benchmark for routing algorithms. Graph Optimization Graph Theory, IP
Steiner Tree Problem Finding a shortest possible network (tree) that connects a given set of required terminal points, potentially by introducing additional intermediate points (Steiner points). Used in network design and VLSI layout. Graph Optimization Graph Theory, Shortest Path
Vertex Cover Problem Finding the smallest set of vertices in a graph such that every edge in the graph has at least one of its endpoints in this set. Important in network security and error correction. Graph Optimization Graph Theory, Set Cover
Shortest Path Problem Finding a path between two vertices (or nodes) in a graph such that the sum of the weights of its constituent edges is minimized. Foundational to navigation systems and network routing. Graph Optimization Graph Theory, Network Flow

IV. Applied & Industry-Specific Optimization Problems

These are real-world challenges that are often modeled using the fundamental paradigms and canonical problems described above, spanning various industries and applications.

Problem Name Description Primary Paradigm Related / Inherits From
Job Shop Scheduling Problem Optimizing the sequence and timing of operations for multiple jobs on multiple machines to minimize total completion time, cost, or maximize throughput, subject to machine and job precedence constraints. Applied Scheduling MILP, CSP
Vehicle Routing Problem (VRP) Finding optimal routes for a fleet of vehicles to deliver goods or services to a set of customers, minimizing total distance, time, or cost, often with capacity and time window constraints. Essential for logistics and transportation. Applied Logistics TSP, MILP
Feature Selection In machine learning, selecting an optimal subset of relevant features from a larger set to improve model performance, reduce overfitting, or speed up computation. A pre-processing step for many ML tasks. Applied Machine Learning BIP, Combinatorial Opt.
Peptide Rotamer Design Optimizing the three-dimensional structure of proteins by selecting the most energetically favorable conformations (rotamers) for amino acid side chains, critical in drug discovery and protein engineering. Applied Biophysics MINLP, CSP
General Scheduling Problem A broad category encompassing problems involving the allocation of resources over time to perform a set of tasks or activities, optimizing criteria like completion time, cost, or resource utilization. Applies to workforce, project, and machine scheduling. Applied Scheduling MILP, CSP, Graph Coloring
Genome Assembly Reconstructing a complete DNA sequence from a large number of short, overlapping DNA fragments (reads), often optimizing for minimum overlaps or maximum contiguity to form the full genome. A core problem in bioinformatics. Applied Bioinformatics Graph Theory, Shortest Path
Traffic Matrix Estimation (TME) Estimating the volume of traffic between different origin-destination pairs in a communication network, often using incomplete measurement data and optimizing for consistency with aggregated flows. Important for network planning. Applied Network Ops LP, Network Flow
Warehouse Inventory Optimization Determining optimal inventory levels, reorder points, and storage strategies within a warehouse to minimize costs (holding, ordering, stockout) while efficiently meeting demand. Key to supply chain management. Applied Supply Chain LP, Stochastic Opt.
Blending Problem Determining the optimal proportions of various ingredients to create a product that meets specific quality requirements at the lowest possible cost, common in industries like food, oil, and chemicals. Often formulated as an LP. Applied Process Opt. LP
Portfolio Optimization Selecting a mix of assets for an investment portfolio to maximize expected return for a given level of risk, or minimize risk for a given expected return. A cornerstone of financial engineering, often using QP and dealing with convex quadratic objectives. Applied Finance QP, Stochastic Opt.
Facility Location Problem Determining the optimal placement of one or more facilities (e.g., warehouses, hospitals, communication towers) to minimize transportation costs, maximize service coverage, or satisfy other objectives. A common strategic planning problem. Applied Logistics MILP, Set Cover
Cutting Stock Problem Determining how to cut larger pieces of stock material (e.g., paper, metal, fabric) into smaller pieces of specified sizes to minimize waste or maximize the number of smaller pieces produced. Often solved with column generation techniques. Applied Industrial Opt. LP, Integer Programming
